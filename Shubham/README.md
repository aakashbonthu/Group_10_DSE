# `# Commodity Price Analysis using Python` 

# `## Project Overview` 

```
This project performs an exploratory data analysis (EDA) on historical commodity
prices using Python. The analysis focuses primarily on **Gold** and **Brent
Crude Oil** prices over time to identify long-term trends, fluctuations, and
significant market events. Various statistical analyses and visualizations were
created to better understand commodity price behavior.
```

# `---` 

# `## Objectives` 

- `Analyze historical Gold and Brent Oil price data.` 

- `Clean and preprocess real-world financial datasets.` 

- `Perform descriptive statistical analysis.` 

- `Visualize long-term price trends.` 

- `Compare commodity prices across different time periods.` 

- `Generate insights from historical market movements.`
